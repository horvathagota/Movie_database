Detailed Description

Ez az osztály alkotja az adatbázist. Itt vannak eltárolva a filmek adatai és a hozzájuk kapcsolódó metódusok is ebben az osztályban vannak definiálva. Az osztály megvalósítja az AbstractTableModel interfészt, hogy a táblázat át tudja venni az adatokat.
